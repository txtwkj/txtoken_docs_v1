## 8.1 HTTP 状态码

| 状态码 | 含义 | 处理建议 |
|--------|------|---------|
| **200** | 成功 | — |
| **400** | 参数错误 | 检查 JSON、枚举值、互斥字段（如 Seedance 场景）；人像/合规类报错见 [§6.2 真人形象审核](/docs/usage/media-api#620-真人形象审核与资产库合规前置) |
| **401** | 认证失败 | 检查 API Key |
| **403** | 权限不足 | 模型白名单 / IP 限制 |
| **429** | 限流 | 降并发、错峰；生图/生视频加大间隔 |
| **500** | 服务器错误 | 上游或渠道异常，稍后重试 |
| **502/503** | 网关/服务不可用 | 上游宕机或过载 |

## 8.2 业务错误码（JSON 内）

| code | 含义 |
|------|------|
| `invalid_api_key` | Key 无效 |
| `insufficient_quota` | 额度不足，联系管理员 |
| `model_not_found` | 模型未开通或名称错误 |
| `do_request_failed` | 上游失败 |
| `rate_limit_exceeded` | 速率限制 |
| `content_policy_violation` | 内容违规 |
| `unknown_parameter` | 传了网关不支持的字段（如部分环境的 `response_format`） |
