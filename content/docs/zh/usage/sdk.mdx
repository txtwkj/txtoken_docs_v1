---
title: SDK 接入指南
description: OpenAI SDK 快速配置
---

使用 OpenAI 官方 SDK 时仅需修改 `base_url` 与 `api_key`：

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://<平台地址>/v1",
    api_key="sk-xxxxxxxxxxxx",
)

# 对话
client.chat.completions.create(model="gpt-4o", messages=[...])

# gpt-image-1.5 / gpt-image-2 文生图
client.images.generate(
    model="gpt-image-1.5",
    prompt="A cute otter",
    size="1024x1024",
    extra_body={"background": "transparent"},
)
```

## 环境变量

```bash
export OPENAI_API_BASE=http://<平台地址>/v1
export OPENAI_API_KEY=sk-xxxxxxxxxxxx
```
